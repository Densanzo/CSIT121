My CSS Learning Process

Published on September 23, 2025 by DS

Lab 3 taught how to utilize the CSS properties to stylize my weblog and take my coding to another level. While I found lab 2 to be quite easy I did the bare minimum and only did what was required, but with previous knowledge on how to use CSS I decided to go a bit beyond what was needed and provided a gif for a background, change hyperlink colors and text decorations in different scenarios, and most of all I had to teach myself again what the difference between IDs and classes were in HTML again. While IDs and classes in other languages are similar in theory, the syntax is a bit different especially if I was using an external style sheet.

W3 schools is very prevalent in the lessons but it only taught me the very basics on how to use gifs, the size was never proportionate to what I needed and I found HTML isn't intuitive enough for me to force myself through the code as I usually do in other languages, either that or I wasn't thinking enough. Since I was using a gif I had to selectively find one that had a uniform color combination but even then in areas where it's lighter or darker, I had to tweak their colors to improve the readabililty while it still fits my theme. Other than that after looking up resources outside of the links provided I was able to look at other lessons on platforms where they asked the same question I had and eventually it worked!

What I Learned

  1. Hyperlinks were taught a little bit in class but after researching more into hyperlinks in CSS there are ALOT more properties that could be used in various ways which was extraordinary!
  2. Background images was the next lesson I tackled and it took a couple more resources outside of the one provided for it to work since I was using a gif, the work was worth it though!
  3. This next lesson had it's basics taught in class, but learning the difference between a class and an ID was necessary so I would know which selector that would be used most often. The selector I used the most was the ID selector.
