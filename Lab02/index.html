My (we)blog for learning HTML this year. While this isn't my first time learning HTML, it has been enough time since I first learned it to find it exciting to create and practicing the language.
From what I knew before and prior experience with Python, Javascript, and C# I am able to enhance and elevate the way I learn by being more efficient.
So this is what I learned this time!

What I Learned

  1. The first thing I learned how to provide breaks with my paragraphs for the website and to break sentences in the actual code to provide better readability for the coding source.
  2. The second thing I learned was using references but when I was using the references and clicking on the links, they redirected me correctly but on the same page,
    losing the other page and not able to look back and forth from the blog and the link so I learned how to add target="_blank" to the references to
    instead redirect the link to a different tab. I think this will be useful in the future when used in the right context.
  3. The third and last lesson I learned for this blog were the semantic elements. While I understood the general usages of tags of "p", "h", "body", and "head" there were other tags
    I remembered that I used before but forgot the context on what they were used for and how to use them so just to broaden my knowledge I learned other elements like summary, aside, and figure!
