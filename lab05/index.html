Lab 2

Published on February 12, 2026 by The Student

Though this isn't my first rodeo to creating a webpage, revisiting the w3schools website helped me recover any information I forgot, which was super helpful! Being provided with apt descriptions of which structure elements were used for and why helped me learn why it was used in that way!

What I learned!

  1. In a link the professor provided from w3schools, it taught the different semantic elements and how and why they were used. I decided before I made this index file, I wanted to use 2 of the semantic elements and decided to use article and article. So in w3schools it explained that the article element could be used for chapters, introductions, news items, and contact info. Further down the page it explained that the article element could be used for forum posts, blog posts, user comments, product cards, and newspaper articles.
  2. When I first clicked a link I created in live preview, it sent me to the provided link's website but on the same page, so if I wanted to look at the same page but read the new website, it wouldn't have been possible unless I went to the previous page and clicked the link again so I decided to look up the code necessary to do what I wanted it to do.The user mwinya explained in order to do it, inside the reference link you would add target="_blank", and it works!
